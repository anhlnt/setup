# Ubuntu 22.04

## Intro
```bash
sudo apt update
sudo apt upgrade
```

## Install Nvidia driver
```bash
sudo apt install nvidia-driver-510
sudo reboot
```

## Install Chrome
```bash
cd ~/Downloads/
wget https://dl.google.com/linux/direct/google-chrome-stable_current_amd64.deb
sudo dpkg -i google-chrome-stable_current_amd64.deb
```
